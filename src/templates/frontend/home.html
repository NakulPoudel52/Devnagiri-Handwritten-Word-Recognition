Nepali Handwritten
Text
Recognition

Handwriting Recognition is the mechanism for converting the handwritten text into a notational representation. It is a special problem in the domain of pattern recognition and machine intelligence.The recognition of texts from the scanned image can have various applications, which are based on optical character recognition. Specially for the Nepali language there are very few researches in the domain of handwriting recognition. Mostly used recognition tools include statistical based recognition tools.
